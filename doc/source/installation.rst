Installation
============
.. toctree::
   :maxdepth: 3
   
   
Prequisites
-----------

* :code:`numpy`


Setup
-----

Install via pip:

.. code-block:: bash

    pip install mpcpy
    
Or: 

* download a `release <https://github.com/BrechtBa/mpcpy/releases>`_
* unzip and cd to the folder
* run :code:`python setup.py install`
